Alarme

Installation de l'appli

  1. Installer l'appli pour Android ou iOS/iPhone.
  2. Prendre rendez-vous avec Sven pour la configuration de l'appli.

Utilisation

Pour armer (activer) ou désarmer (désactiver) l'alarme, il faut taper sur PAP alarme dans la liste, ensuite sur l'icône cadenas correspondante. (La troisième icône Voir permet de vérifier l'état sans le modifier.)

L'appli crée un texto au système d'alarme qu'il faut envoyer. Le système répond avec un texto de confirmation qui n'est malheureusement pas traduit en Français.

System scharf – Le système est armé (actif).

System unscharf – Le système est désarmé (inactif).
